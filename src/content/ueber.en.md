# About StoryboardDesigner

## Purpose

StoryboardDesigner is an educational web app that lets students plan storyboards
for film, photo story or speech. It is built following the haak3 web app
standard for classroom-ready, privacy-conscious tools.

## How it works

The app runs entirely in your browser. Text and images are processed locally and
are not transmitted to a server. Projects can be saved to your device as a
`.storyboard` file (a ZIP containing `data.json` and images) and reloaded later.
The current state is also saved automatically in the local browser database
(IndexedDB).

Accessibility features include visible labels, keyboard operation, a dark and a
high-contrast mode, adjustable font size, and image descriptions (alt text).
